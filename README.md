# Telegram Bot Manager - Deteksi Pesan Duplikat

Website untuk mengelola bot Telegram yang mendeteksi pesan duplikat dalam grup.

## Fitur

- Deteksi pesan duplikat dalam grup Telegram
- Pencatatan sejarah pesan dengan tanggal dan pengirim
- Dashboard untuk memantau aktivitas bot
- Deploy mudah melalui Vercel
- Support multiple bot instances

## Cara Menggunakan

1. Dapatkan token bot dari [@BotFather](https://t.me/BotFather)
2. Deploy aplikasi ini ke Vercel
3. Masukkan token bot di dashboard
4. Tambahkan bot ke grup Telegram Anda
5. Berikan izin admin kepada bot untuk membaca pesan
